VIGA: Vision-as-Inverse-Graphics
as a VLM Coding Agent

A unified framework that leverages Vision-Language Models to understand and manipulate 3D scenes through inverse graphics and code generation.

Abstract

We present VIGA, an execution-grounded, analysis-by-synthesis code agent that reconstructs real images as editable, executable scene programs in graphics engines. Rather than generating scene code in a single pass, VIGA alternates between Generator and Verifier roles: it writes code, executes it in a graphics engine, renders the scene from multiple viewpoints, identifies discrepancies with the input image, and iteratively refines the program. Operating entirely zero-shot without fine-tuning, VIGA leverages an evolving contextual memory that accumulates plans, code diffs, verification feedback, and render history to progressively transform imprecise attempts into coherent programs. Beyond 3D reconstruction, the framework enables programmatic scene editing, 4D dynamic reconstruction, and structured 2D layout synthesis. We introduce BlenderBench to evaluate reconstruction fidelity and agentic behaviors including multi-view spatial consistency, discrepancy localization, and sustained multi-modal tool use.

Method

Pipeline Overview

Given any input image, VIGA can reconstruct the 3D scene in Blender through an analysis-by-synthesis process.
First, a generation agent proposes executable code that modifies the scene.
After executing the code, a verification agent observes the updated scene and produces feedback for refinement.
This feedback is then returned to the generator to guide the next round of generation.

Quantitative Results

Comprehensive quantitative evaluation demonstrating VIGA's effectiveness across multiple benchmarks:
BlenderGym for single-step graphics editing (+32.65% average improvement),
AutoPresent for slide generation (+117.17% average improvement), and
BlenderBench for complex multi-step 3D reconstruction (+80.24% average improvement).
